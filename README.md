# Tic Tac Toe Game

WELCOME TO THE TIC TAC TOE GAME WEBSITE.



## Description

This is a web-based implementation of the classic game "Tic Tac Toe". The game allows two players to play turn by turn using X and O. The objective is to place three marks in a row (horizontal, vertical, or diagonal) to win the game.



## Features

➤ Simple and intuitive user interface.

➤ Two-player gameplay (Player X and Player O).

➤ Automatic winner detection.

➤ Draw detection system.

➤ Reset game option.

➤ New game button available.



## Usage

➤ Open the game in your browser.

➤ Player X starts the game first.

➤ Click on any box to place your mark.

➤ Players take turns one by one.

➤ The game will automatically detect the winner.

➤ If no player wins, the game ends in a draw.

➤ Use Reset or New Game to play again.



## Game Logic

➤ Game Start: The game begins automatically when the page loads.

➤ Player Turns: Players take turns marking empty boxes.

➤ Winner Check: After each move, the game checks all winning patterns.

➤ Win Condition: If a player matches a pattern, they win.

➤ Draw Condition: If all boxes are filled with no winner, the game ends in a draw.



## Technology Used

➤ HTML: For the structure of the game.

➤ CSS: For styling and layout.

➤ JavaScript: For game logic and interactivity.
